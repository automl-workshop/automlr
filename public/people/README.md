# Committee photos

Drop headshots here to fill the People section. Each avatar loads
`/people/<slug>.jpg` automatically; until a file exists, the card shows the
person's initials. No code changes needed — just add the file with the exact
name below.

**Format:** square JPG (or swap the extension in `index.html` if you prefer),
ideally ~400×400px, face centered. Keep files reasonably small (<150 KB).
